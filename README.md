# Simon Game

A classic memory-based Simon Game recreated using React.  
The player must repeat an increasingly long sequence of colors generated by the game.

## Overview

This project is a modern implementation of the traditional Simon Game.  
It focuses on state management, component architecture, and interactive UI behavior using React.

The application generates a random color sequence that the user must reproduce correctly.  
Each successful round increases the difficulty by adding a new step.

## Demo

Live version:  
https://simon-game-react-nine.vercel.app/

## Features

- Interactive color pads with visual feedback
- Sound effects for each color
- Progressive difficulty (level system)
- Real-time input validation
- Game over state handling
- Keyboard and button start support
- Responsive layout
- Component-based architecture

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- CSS3

## Architecture

The UI is structured using reusable React components:

- `Header` – Displays game title and status
- `GameBoard` – Renders the color pads
- `Pad` – Reusable pad component
- `StartButton` – Game control button
- `Footer` – Credits section

Game logic is handled inside the main `App` component using React Hooks (`useState`, `useEffect`).

## Game Logic

- A random color is added to the sequence at each level
- The full sequence is replayed visually and audibly
- User input is validated step-by-step
- Incorrect input triggers the game over state
- Correct completion advances to the next level

## Visão Geral

Este projeto é uma implementação moderna do tradicional Simon Game.  
O foco está no gerenciamento de estado, arquitetura em componentes e comportamento interativo da interface usando React.

A aplicação gera uma sequência aleatória de cores que o usuário deve reproduzir.  
A cada rodada bem-sucedida, a dificuldade aumenta com a adição de um novo passo.

- Validação em tempo real

## Arquitetura

A interface foi estruturada em componentes reutilizáveis:

- `Pad` – Componente reutilizável dos botões
- `StartButton` – Botão de controle do jogo

## Lógica do Jogo

- Um erro ativa o estado de game over
- Sequência correta avança o nível
